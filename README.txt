1. WwsoftBaseAuthenticator
this class provides generic methods for creating JWT key, creating JWT token, and parse a given JWT token.

2. WwsoftAuthenticator 
This class provides methods second level methods for creating JWT key, creating JWT token, and parse a given JWT token.

3. Three JUNIT-based test programs have been created.

WwsoftAuthenticatorCreateJWTKey - create a JWT key
WwsoftAuthenticatorCreateToken - create a JWT token
WwsoftAuthenticatorUseToken - use the JWT token
